A comprehensive patient management portal built with React.js and Node.js for modern healthcare facilities.

## 🏥 Features

- **Patient Dashboard** - View medical records, appointments, and test results
- **Appointment Booking** - Schedule appointments with available doctors
- **Medical Records** - Secure access to patient medical history
- **Doctor Communication** - Direct messaging between patients and healthcare providers
- **Prescription Management** - View and manage prescriptions
- **Insurance Integration** - Insurance information and claims tracking

## 🛠️ Tech Stack

- **Frontend:** React.js, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT
- **Security:** HIPAA-compliant data encryption

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/yourusername/healthcare-patient-portal.git

# Install dependencies
npm install

# Start development server
npm start

# Start backend server
npm run server
```
